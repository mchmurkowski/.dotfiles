# .dotfiles

My personal dotfiles for Linux and WSL.

Contains configuration for:

- alacritty
- bash
- dwm[^1]
- editorconfig
- git
- nvim
- starship
- tmux
- xinit

# Usage

Dotfiles are managed with `stow`:

```bash
man stow
```

[^1]: dwm patches: [attachbottom](https://dwm.suckless.org/patches/attachbottom/), [fullgaps](https://dwm.suckless.org/patches/fullgaps/).
